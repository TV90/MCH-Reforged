## ✨ 项目简介
**MCHeli-Reforged（MCH-R）** 是经典 **MC Helicopter (MCHeli)** 模组的现代化重制/强化分支，聚焦“更真实的战斗体验”。在保留原有飞机、直升机、坦克、武器与内容包生态的基础上，引入大量新机制，并对原有代码做了重构优化。对应仓库当前采用 GPL-3.0 开源许可。
